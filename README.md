# BoxingCI

Biblioteca visual de boxeo en español.

## Estructura

```text
BoxingCI/
├── index.html
├── assets/
│   └── style.css
├── herramientas/
│   └── teoria-de-las-lineas/
│       └── index.html
├── biblioteca/
│   ├── rangos/
│   │   └── index.html
│   ├── golpes/
│   │   └── index.html
│   └── defensa/
│       └── index.html
```

## Uso

1. Sube todo al root del repo.
2. Reemplaza `herramientas/teoria-de-las-lineas/index.html` con tu visualizador.
3. Activa GitHub Pages desde `Settings → Pages → Deploy from a branch → main → /root`.
